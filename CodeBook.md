# Codebook

## Dataset
The datasets included in this repository have been obtained after processing the "Human Activity Recognition Using Smartphones Data Set" from the University of California, Irvine. The dataset and its description is available [here](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones).

## Transformations
The transformations to the dataset were originally defined in the [class project](https://class.coursera.org/getdata-010/human_grading/view/courses/973497/assessments/3/submissions) for Coursera's Getting and Cleaning Data course. Here is a summary of these transformations:

1. The training and the test sets were merged to create one data set.
2. The measurements on the mean and standard deviation for each measurement were extracted. 
3. Activity names were modified to be descriptive.
4. The data set was appropriately labeled with descriptive variable names. 
5. From the data set in step 4, a second, independent tidy data set was created with the average of each variable for each activity and each subject.

These transformations were performed in R and are detailed in [run_analysis.R](./run_analysis.R).

## Variables
The datasets created in steps 4 and 5 described above are stored in [mean_and_std.txt](./mean_and_std.txt) and [averages.txt](./averages.txt), respectively.
